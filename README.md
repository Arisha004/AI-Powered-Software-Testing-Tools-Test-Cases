# AI in Software Testing

## Overview
This repository showcases my practical learning from the **AI in Software Testing** course. It highlights my understanding of how artificial intelligence can support and improve software testing activities, including manual testing and AI-assisted automation.

The work in this repository reflects hands-on experience with modern AI-based testing tools and the use of AI assistants for writing and refining test cases.

---

## Skills Gained
- Understanding AI concepts applied to software testing
- Manual test case design using AI assistance
- Identifying functional, negative, and edge test scenarios
- Performance and quality analysis
- Exposure to AI-driven test automation tools

## Tools and Technologies
